# [Front-end]()

Este projeto foi criado utilizando _**React Native Expo**_.

---
### ✦ Pré requisitos

Antes de começar, certifique-se de ter instalado:

- _**[Node.js](https://nodejs.org/)**_

---
### ✦ Instalando as dependências

   ```bash
   npm install
   ```
---
### ✦ Rodando o projeto

   ```bash
   npx expo start
   ```
Após isso, escolha como abrir o app:


#### **Celular físico**

Um **QR Code** será exibido no terminal, abra o app **Expo Go** no seu celular e escaneie o código. 
Certifique-se de que o celular e o computador estejam na **mesma rede Wi-fi.**

#### _**Emulador Android / IOS**_ 

Com o emulador em execução, pressione **' a '** para **android** ou **' i '** para **IOS** no terminal. 
O app abrirá automaticamente, sem precisar de um QR Code.

---
## ✦ Iniciando do zero 

Se quiser limpar o projeto inicial e começar do zero:

```bash
npm run reset-project
```
Esse comando moverá o código de exemplo para **' app-example/ '** e criará um diretório **' app/ '** limpo.
