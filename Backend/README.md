# [Back-end]()

Este projeto foi criado utilizando _**Python**_.

---
###  ✦ Pré requisitos

Antes de começar, certifique-se de ter instalado:

- _**[Python](https://www.python.org/)**_ 

---
### ✦ Instalando as dependências

   ```bash
   pip install -r requirements.txt
   ```
---
### ✦ Rodando o projeto

Primeiro verifique no terminal se você está na  pasta _**main**_ e  depois execute esse comando:

   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload
   ```
